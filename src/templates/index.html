COVID19 API

Simple covid-19 statistics api

GET /world

Get all covid19 statistics

Example: /world 
        
    {
        "active_cases": 18404546,
        "critical_cases": 109628,
        "new_cases": 329867,
        "new_deaths": 5313,
        "total_cases": 143874047,
        "total_deaths": 3062294,
        "total_recovered": 122407207
    }
